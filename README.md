Usage

```bash
# автивация виртуальной среды
source project_env/bin/activate

# установка пакета и обновление источника зависимостей
pip install name
pip freeze > requirements.txt

# установка всех зависимостей из источника
pip install -r requirements.txt

# запуск скрипта
python ./script.py

# деактивация виртуальной среды
deactivate
```
